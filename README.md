# dotfiles

My dotfiles, managed with [chezmoi](https://www.chezmoi.io/).

## What's inside

- **fish** shell config
- **neovim** config
- **kitty** terminal
- **zellij** multiplexer
- **lazygit**, **gh** CLI
- **karabiner** key remaps
- **starship** prompt
- **Brewfile** with all packages (brew, casks, go tools, cargo)
- **ssh** config

## Setup

```sh
chezmoi init --apply https://github.com/kryptamine/dotfiles
```
